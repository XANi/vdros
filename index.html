Virtualisation Debug/Recovery OS

This is a simple Linux image with a bunch of useful tools for debugging and fixing VM environments, including network (nmap and friends), performance (a bunch of benchmarking tools) and rescue (few tools to mount and repair existing filesystem).

It was inspired by cirros Cirros, but expanded to provide for functionality sysadmin needs, and remove unnecesary waste of time (like typing exactly same default passowrd when logging on console.

Building (Debian/Ubuntu)

$ sudo apt-get install gcc-multilib build-essential libncurses5-dev git bzip2
$ ./build.sh

if all goes correct, images should show up in images ( ext/buildroot/output/images)

Usage

By default it spawns passwordless root shell on every console + ttyS0. Kernel output goes on ttyS0 and tty1 so if you use Openstack you should see it in Log tab.

Networking is defined to spawn DHCP on every ethernet interface (en*)
